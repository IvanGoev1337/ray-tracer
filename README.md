# Ray Tracer

## Overview

Ray Tracer is a C++ graphics project that implements the fundamentals of ray tracing using custom mathematical primitives, ray logic, and scene geometry. The project is intended for educational purposes and demonstrates how rendering algorithms can be built from scratch.

It is a good example of low-level work with rendering logic, object intersections, and scene representation.

## Features

- Custom ray tracing pipeline
- Scene rendering based on mathematical ray/object intersections
- Custom vector and geometry abstractions
- Modular code structure for extending shapes and rendering logic
- OpenGL-based window integration for visualization

## Technologies

- C++
- OpenGL
- GLFW
- GLEW
- Visual Studio
